Clean CaDET

Project by The Science Fund of the Republic of Serbia

Our Project

The main objective of this project is to lower the cost of software development and increase its overall quality. We shall address this objective by utilizing artificial intelligence (AI) algorithms to detect code smells in software code. We shall encapsulate the AI algorithms in a usable open-source tool, to support software developers in their work and help them discover unclean code as they write it. We shall supplement the detection mechanism with educational resources to help the developer understand the nature of the smell and the best way to resolve it. Finally, we shall enhance the tool with an AI-based tutoring system, to offer personalized advice and educational material to the developer and ensure their efficient learning. We call the proposed tool the Clean Code and Design Educational Tool (Clean CaDET). eFor Clean CaDET to effectively fulfill the main objective of the project, multiple sub-objectives need to be reached, each tackling a specific research problem:

  1. Develop a set of automatic AI-based code smell detection techniques, addressing the most widespread and severe code smells.

  2. Collect a novel dataset of software artifacts for training AI-based code smell detection algorithms.

  3. Develop an AI-based educational system for delivering personalized multimedia to the learner, detailing the nature and impact of the smell, as well as the recommended refactoring solution.

  4. Develop a usable open-source tool, integrating the developed detection techniques and educational system, to offer automatic smell detection on custom code and learning material recommendation.

The purpose of Clean CaDET is to automatically detect code smells in the developer’s code and offer personalized advice on how best to deal with the detected smells, thus training the developer to write code of higher quality. The Developer uses their integrated development environment (IDE) to write code (1) for their software solution. The Clean CaDET Plugin (C2Plugin) processes the code and sends its representation (2) to the remote Clean CaDET Application (C2Application). The Clean Code Analyzer module executes a series of AI-based smell detection techniques to detect different code smells in the developed code. The Clean Code Analyzer also contains a dataset of software artifacts, used to train the AI-based smell detection techniques. Then, the Analyzer sends the discovered code smells (3) to the Smart Tutor module. This module represents an AI-based educational system that personalizes the learning experience using a recommender system. It selects the most suitable content (e.g., text, illustration, video) from the repository of educational material to facilitate the Developer’s learning. Once selected, Clean CaDET returns the educational multimedia (4) to the C2Plugin, which displays the information to the Developer (5).
